How to run locally
- need to have Java and Postgresql database
- have the variables for URL, Username and Password
- run ./mvnw spring-boot:run
- checkout port 8080, http://localhost:8080

How to Deploy
- Push to GitHub
- created new Web Service
- connected repo from git
- added postgresql database and setup environments
- setup docker as well
- will then auto deploy after every git push.
